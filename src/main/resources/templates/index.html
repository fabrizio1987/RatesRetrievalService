Step 1 - Job Configuration

Configure the file ratesProcessJob.xml inside the META-INF/batch-jobs.

Specifically, configure the following properties:
  • <name="scan_directory" value="ABSOLUTE_PATH_FOLDER"/> contains the folder path to scan.
  • <property name="archive_directory" value="ABSOLUTE_PATH_FOLDER"> contains the folder path where to store the files after being processed.
  • <property name="failed_directory" value="ABSOLUTE_PATH_FOLDER"/> contains the folder path where will be store the files with records that has been not possibile to process.

Step 2 - Copy files into the directory

Before starting the job, remember to copy the files to process into the folder you've set int the properties scan_directory of the ratesProcessJob.xml configuration file.
